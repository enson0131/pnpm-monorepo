# pnpm-monorepo

基于 pnpm workspace 实践的多包管理项目

## apps - 业务系统目录

- backend - 服务端业务
- frontend - 前端业务

## packages - 基础包目录

- chart - 图表库
- cli - 脚手架
- tools - 工具库
- ui - 组件库
